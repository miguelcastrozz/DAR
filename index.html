Biografía

Nací en Bogotá. Desde temprana edad, mostré un gran interés en la tecnología. Asisto a la Universidad Internacional de La Rioja y estoy estudiando ingeniera informatica. Comencé mi carrera profesional en SeaHubs, donde logré experiencias significativas. Actualmente, sigo viviendo en Bogotá y continúo con actividades relacionadas con la ciencia de datos y machine learning.

Experiencia

Cada experiencia es una oportunidad de aprendizaje. En esta sección, he compartido algunas de las experiencias más significativas de mi carrera, desde roles de liderazgo hasta proyectos desafiantes. Verás cómo cada una de estas experiencias ha contribuido a mi desarrollo profesional, ayudándome a crecer como profesional y como persona.

Empresa: Sol4IT

Puesto: Consultor asociado de Oracle NetSuite

Fechas: 2023 - 2024

Descripción: En este rol como consultor, he liderado y optimizado diferentes proyectos, desde sencillos hasta complejos, donde analizamos los requerimientos de negocios para garantizar la satisfacción de los mismos. Este rol ha fortalecido mis habilidades técnicas y blandas, permitiéndome identificar requerimientos de negocio, diseñar procesos basados en las necesidades del negocio, validar el funcionamiento implementado mediante pruebas exhaustivas, realizar actualizaciones y configuraciones de NetSuite, y resolver inquietudes durante y después de las implementaciones del negocio, entre otros. Las funciones principales que desempeñaba eran las siguientes.

Responsabilidades: Liderar el equipo de desarrollo, revisar el código, etc.

  • Facturación electrónica: Implementé y mantuve soluciones de facturación electrónica en NetSuite para mejorar la eficiencia, reducir el tiempo en gran medida para el negocio y aumentar la precisión de los procesos de facturación con clientes.
  • SuiteScript: Desarrollé y depuré scripts SuiteScript para configurar, automatizar y personalizar los procesos de negocio en NetSuite.
  • Plantillas: Creé y personalicé plantillas para mejorar la presentación y distribución de documentos comerciales.

Tecnologías: API REST - CSS 5 - FreeMarker - HTML 3 / 5 - JavaScript ES5 / ES6 - JSON - Node - SuiteCloud - SuiteScript 1.0 / 2.1 - TypeScript - XML
